# Roadmap: NBA Player Analysis System

## Core Philosophy

Each dimension is an independent ranking with its own methodology.
Composite ranking comes LAST, after all dimensions are solid.

## Dimensions

### Done
- [x] **Scoring** - PPG x TS+, FT penalty, playoff weight, era scarcity
- [x] **Offensive Impact** - Ridge on O-DPM, era Z-score adjusted

### Next Up
- [ ] **Playmaking** - AST%, Box Creation, potential assists, assist quality
- [ ] **Defense** - D-DPM, DBPM, defensive rebounds, steals, blocks
- [ ] **Rebounding** - ORB (second chances) vs DRB (ending possessions), separate analysis
- [ ] **Efficiency IQ** - Shot selection, turnover rate, USG vs TS tradeoff

### Final
- [ ] **Position-adjusted ranking** - Compare within position groups
- [ ] **Composite GOAT ranking** - Multi-dimension aggregate (only after all above)

## Bonus Ideas (for fun)
- [ ] Offensive style classification (3pt shooter / mid-range / paint / FT-driven)
- [ ] Playoff clutch index (performance increase in elimination games)
- [ ] Solo carry ability (On/Off + teammate quality adjustment)
- [ ] Scoring consistency (game-to-game variance)
- [ ] Player evolution (early career vs prime vs late career)
